Tee Time BRS In plain English

This is the one place your golf group can see dates, sign up, and stay in sync—without text storms.

What it does for you

  • Keeps a live calendar for tee times and team events so everyone knows when and where.
  • Sends the right emails: new events, updates, cancellations, empty-tee reminders, and (if you choose) a note to the club when you delete a tee time.
  • Makes moving golfers easy: add, remove, move between times/teams, or park them on a “Maybe” list.
  • Shows who changed what with an admin code for deletes, so accidents and mysteries are rare.
  • Includes extras your group uses: Myrtle trip tracker.

Everyday tasks

  • Create events quickly: Pick course, date, first tee time, and how many times (auto-spaced). For teams, pick team size and start type.
  • Auto-create from email: Forward course confirmations to teetime@xenailexou.resend.app. It reads the email, makes the event, and sets tee-time count. Cancellation emails remove the match.
  • Manage tee times: Add/remove/rename times; when deleting, enter the admin code and choose whether to notify the club (CC your admin).
  • Manage teams: Shotgun or staggered; teams auto-named; team size enforced.
  • Players: Add or move golfers; remove if needed; keep overflow in Maybe.

Helpful automation

  • Club notice on delete: If you choose, the course gets an email (with CC) asking them to remove the tee time from their system.
  • Course info: Search shows location, phone, website, and par/holes.
  • Weather glance: Quick icon and temps for events within 16 days.

Admin safety

  • Admin code required to delete events/tee times.
  • Audit log shows adds/removes/moves.
  • Admin page manages subscribers and reminders.

Special pages

  • Myrtle Trip: /myrtle-trip-2026.html for trip details.

How to roll it out

  • Share the site and your access code.
  • Ask players to click “Subscribe” so they get emails instead of texting you.
  • Tell them: pick a date, pick an event, add yourself—no spreadsheets or reply-all.

Captain’s daily playbook

  • Create or import the event; check times look right.
  • Watch sign-ups; move players to balance groups; use Maybe for overflow.
  • If a tee is cancelled, delete it, enter the admin code, and choose “notify club.”
  • Check audit log if something seems off; send reminders from the admin page if a tee is empty.
