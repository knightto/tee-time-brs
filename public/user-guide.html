Getting Started

  • Enter your group access code to view the calendar and events for that group.
  • Select any date to see events; use the “New Tee-Time Event” or “New Team Event” buttons to add one.

Create Tee-Time Events

  • Manual: Pick course, date, first tee time, notes, and choose how many tee times (9-minute spacing).
  • Email import: Course emails sent to teetime@xenailexou.resend.app (from no-reply@foreupsoftware.com or your Gmail) are auto-parsed. Subjects like “Tee Time Reservation Confirmation” create events; golfer count ÷ 4 sets how many tee times are generated.

Cancel via Email

  • Subjects containing “Reservation was cancelled” delete the matching event and notify subscribers.

Team Events

  • Shotgun or Tee-Time start; set team size (2–4). Three teams created by default; add more anytime.

Player Management

  • Add/remove/move players with capacity checks; “Maybe” list for interested golfers.

Course Search & Info

  • Live Golf API search with inline location, phone, website, and holes/par; cached for speed.

Weather

  • Auto weather icons, temps, and descriptions for events within 16 days.

Notifications

  • Subscribers get emails for new events, updates, cancellations, reminders for empty tee times, and admin alerts. Unsubscribe links are included.

Admin & Extras

  • Admin dashboard: manage subscribers, send reminders, verify course data, view stats.
  • Audit log shows who was added/removed/moved and when.
  • PWA support with offline caching and golf favicon.
