Tee Time BRS User Guide

Use this page for step-by-step usage. For a full capability list, open the Features page.

Quick Start

  • Open the main page and click Subscribe to get email updates.
  • Pick a date on the calendar to view events for that day.
  • Create an event with New Tee-Time Event or New Team Event.
  • Open an event and use Actions to add times/teams, add players, and share calendar links.
  • Use Refresh any time to force a fresh data pull.

Top Bar Options

  • Info menu: User Guide, Features, Blue Ridge Outings, Weekend Game Rules, Handicaps.
  • Operations Guide: live reference page for Render, Resend, email addresses, and workflow details.
  • Subscribe: saves your email for event notifications.

Calendar Options

  • Previous/Next month: arrow buttons in the calendar header.
  • Select day: tap any date cell to load events for that day.
  • Add tee times to your calendar for this month: downloads a monthly ICS file.
  • Request a Tee Time for Blue Ridge Shadows: sends a date/time request to the club.
  • New Tee-Time Event / New Team Event: create events directly from the sidebar.

Event Card Options

  • Actions toggle: show/hide event action buttons.
  • Add Existing Time (tee-time events) or Add Team (team events).
  • Request Club Time: sends an additional tee-time request for that event date.
  • Pairings: suggests and applies handicap-balanced groups for tee-time events.
  • Edit / Delete: edit event details, or delete with admin code.
  • Google / ICS: add a single event to calendar apps.
  • Audit button (clipboard icon): view event change history.

Tee Time and Team Row Options

  • Add Player: add a golfer to a tee time/team.
  • Check In All: mark or clear check-in for the entire tee/team.
  • Individual actions: check in, move, or remove each golfer.
  • Edit team name: rename teams in team events.
  • Remove tee/team: delete a slot (with optional club notification path when prompted).

Maybe List Options

  • + Interested: add someone to the maybe list.
  • Fill Spot: promote someone from maybe to an open spot.
  • Remove (x): remove a maybe entry.

Status and Visual Cues

  • Weather summary: day icon, low/high, and rain percentage when relevant.
  • Full tee rows: muted/grey styling with full indicator.
  • Empty tee rows inside 3 days: bright red warning style.
  • Calendar marker inside 3 days: red golf marker on affected dates.

Email Import Workflow

  • Forward tee-time confirmation emails to teetime@xenailexou.resend.app.
  • The system parses date/time/course and creates the event automatically.
  • Cancellation emails remove matching events when detected.

Tips For Captains

  • Create events early, then use Maybe + Fill Spot while the list settles.
  • Use Pairings close to event day for better handicap balancing.
  • Use Check In All and individual check-in to track arrivals quickly.
  • Use the admin audit view if any roster change looks unexpected.
