Tee Time BRS Features

This page lists options and capabilities. For usage steps, open the User Guide.

Back to Tee Times Open User Guide Open Admin Page

User Options

  • Info navigation: User Guide, Features, Blue Ridge Outings, Weekend Rules, Handicaps.
  • Subscribe: opt in for event notification emails.
  • Calendar navigation: month switching, date selection, and event list by day.
  • Monthly calendar export: one-click ICS download for the visible month.
  • Calendar tee-time request: send a date/time request to the club.
  • Event creation: tee-time events and team events with appropriate settings.
  • Event actions: add time/team, request club time, pairings, edit, delete, Google, ICS, audit.
  • Player actions: add, move, remove, individual check-in, and check-in all.
  • Maybe list: interested list, fill spot, and remove entries.
  • Course details: city/state, phone, website, holes/par when available.
  • Weather display: forecast icon, low/high temps, rain chance display.
  • Visual urgency: red empty tee rows and red calendar markers within 3 days.
  • Fresh mobile behavior: automatic refresh on resume/focus/online/pageshow.
  • Email import: auto-create/cancel events from forwarded club emails.
  • Calendar reminders in ICS: 3-day and 1-day reminders included.

Admin Functions

  • Global notifications toggle: enable/disable system-wide subscriber emails.
  • Automated scheduler toggle: pause or resume all scheduled jobs.
  • Scheduled email rule toggles: 4 PM Brian empty-tee alert, 5 PM 48-hour reminder, 5 PM 24-hour reminder, 5 PM nearly-full alert, 6-hour admin empty-tee alerts.
  • Quick actions: check empty tees, refresh all weather, verify course data, message all subscribers.
  • Subscriber management: view subscriber list and delete subscribers.
  • Tee-time change log: audit adds, edits, deletes, moves, and check-ins.
  • Protected actions via admin code: event delete, tee/time delete, club-notify delete path.
  • Manual admin endpoints: reminder runs, Brian empty-tee alert run, empty-tee report, course verification.
